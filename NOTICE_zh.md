# 发布说明

这个目录包含：

- 基于 NIST CONTAM 的中文化程序与帮助文件
- 已经生成好的 `ContamHelp.chm`
- 中文帮助 HTML 源码
- 汉化复现材料

## 来源说明

- `CONTAM` 主程序资源中标注 `Public Domain Software`
- 中文帮助中的免责声明页说明：CONTAM 属于美国联邦政府作品，可自由再分发和修改，但衍生版本应注明源自原软件，修改版本应注明已修改

## 当前改动

- 增加了中文资源文本
- 增加了中文帮助文件
- 增加了命令行工具的双击包装器
- 未修改原始数值求解器的算法逻辑

## 分发建议

- 公开发布时，保留本文件和 `README_zh.md`
- 如果你继续修改界面或帮助，建议在仓库简介或发行说明里注明“基于 NIST CONTAM 的中文化修改版”
- 不要把微软 HTML Help Workshop 编译器一起公开分发；需要重建 `CHM` 时，让使用者自行下载安装
