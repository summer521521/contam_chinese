# NOTICE

## Project Identity

This repository is an unofficial Chinese-localized modified distribution based on NIST CONTAM. The public repository preserves localization assets, documentation, and release scripts, while runnable binaries are distributed through GitHub Releases.

The repository homepage, release page, and package notes retain the following statements:

- `Based on CONTAM developed by NIST.`
- `This is an unofficial Chinese-localized modified distribution.`

## Upstream Source

Original software:

- Name: `CONTAM`
- Publisher: `NIST`
- Official download page: <https://www.nist.gov/el/energy-and-environment-division-73200/nist-multizone-modeling/software/contam/download>
- Software page with notice information: <https://www.nist.gov/services-resources/software/contam>
- Reference document: <https://doi.org/10.6028/NIST.TN.1887r1>

According to the disclaimer in the official CONTAM help and documentation:

- Under U.S. law, the upstream CONTAM software is a work of the federal government.
- The software may be redistributed and modified.
- Derivative works should acknowledge that they are derived from CONTAM.
- Modified versions should clearly state that they have been modified.

This NOTICE preserves the source notice and modification notice for this project.

As of `2026-03-16`, the latest CONTAM release listed on the official NIST download page is `3.4.0.8`, released on `2026-01-08`. The same page states that the bundled `ContamX` version is `3.4.0.3`.

## Modifications In This Repository

This repository includes the following major modifications or additions:

- Chinese localization of `contamw3.exe` interface resources
- Chinese help file `ContamHelp.chm`
- Chinese help HTML source and build materials
- Scripts used to build the Chinese help file
- Public release documentation such as the README, changelog, and release template

This NOTICE does not declare changes to the original numerical solving algorithms.
