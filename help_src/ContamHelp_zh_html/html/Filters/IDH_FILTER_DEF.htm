过滤器属性

过滤器数量： 该编号由 ContamW 自动分配，并且可以在您向项目添加过滤器或从项目中删除过滤器时更改。每次保存项目时，都会重新分配过滤器编号。

尽管这是 CONTAM 使用的实际过滤器编号，但在显示过滤器累积结果时将使用不同的编号 污染物摘要文件 （.csm 文件）。该文件将根据与其关联的建筑组件来显示过滤器。例如，与气流路径和空气处理系统供应和返回点相关的过滤器将通过路径编号和字母“p”来引用，以指示它是路径过滤器。与管道相关的过滤器将通过管道段编号和字母“d”或端子编号和端子字母“t”来引用。

过滤器 元素

姓名： < 必填 > 您可以从名称列表中包含的过滤器元素中选择先前定义的过滤器元素。单击“新元素”按钮定义新的过滤器元素。要查看或修改过滤器元素数据，请单击“编辑元素”按钮。您还可以访问 CONTAM 库管理器，通过单击“库”按钮从库文件导入过滤器元素（请参阅 使用数据和库 ）。每个过滤器都必须有一个与其关联的过滤器元件。

描述 : 显示名称字段中当前显示的过滤器元素的描述。

类型： 显示“名称”字段中当前显示的过滤器的过滤器元素类型。

初始过滤器加载

总相对负载： 设置过滤器相对于过滤器质量的初始负载。可以将其设置为零以表示模拟开始时的“干净”过滤器；否则设置该值以确定模拟开始时过滤器上累积的污染物总质量（以过滤器质量的百分比表示）。相对负载仅用于气体过滤器元件，该元件是目前唯一将效率描述为过滤器负载函数的元件。
