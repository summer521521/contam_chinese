创建 WPC 文件对话框

创建 WPC 文件对话框是一个可以帮助创建 WPC 及关联文件的工具。无论您是使用外部编辑器创建 WPC 文本文件还是使用 EWC File Converter 。通过以下方式访问此对话框 Weather → Create WPC File... 菜单项。

注意： WPC 文件可以在模拟过程中与瞬态天气和瞬态污染物文件结合使用，如下所示：

  • 如果使用 WPC 文件中的风压，则不能使用 WTH 文件。
  • 如果使用 WPC 文件中的污染物浓度，则不得使用 CTM 文件。

Name of WPC File to create: 这是使用以下命令创建的 WPC 文件的名称 EWC File Converter。您可以使用 Browse... 按钮选择现有文件，或输入新名称。

WPC Description: 输入您想要的描述 EWC File Converter 创建新的 WPC 文件时写入。

EWC File Name: The External Wind Pressure and Contaminant (EWC) 文件包含外部污染物浓度和压力， EWC File Converter （必须为您的特定应用程序开发）将转换为 WPC 文件。使用 Browse... 按钮选择现有的 EWC 文件。

WPC File Creation Program: 使用 Browse... 按钮选择用于创建 WPC 文件的程序，例如 EWC File Converter.

坐标变换数据

这些参数可以由 EWC File Converter 建立 EWC 文件和 CONTAM 项目文件的坐标之间的关系。如果 PRJ 和 EWC 文件的坐标系一致，则不需要转换。这些值通过 ContamW 存储在 PLD 文件中。

Origin (X, Y and Z): CONTAM PRJ 文件的来源相对于 EWC 文件的来源的位置。

Rotation Angle: EWC 坐标系的 x 轴和 y 轴绕 z 轴旋转，以与 CONTAM 坐标系的 x 轴和 y 轴对齐。逆时针方向被认为是正方向。

转换容差  

这些是公差 EWC File Converter 可能用于确定 EWC 文件和 PLD 文件中的信息必须匹配的程度，以便解析污染物（种类）和位置数据。这些值通过 ContamW 存储在 PLD 文件中。

Species: 物种可以通过其摩尔质量来解析，即 ContamW 和 WPC 文件中定义的每个物种的摩尔质量的差异不得超过此数量才能被视为相同物种。

Location: 位置可以通过距离来解析，距离由 EWC File Converter ，例如，到网格单元中心的距离。 PLD 文件中的单位为米。

日期和时间

这些参数可以由 EWC File Converter 确定写入 WPC 文件的日期和时间值，以便于根据 CONTAM 的时间表进行工作。这些值通过 ContamW 存储在 PLD 文件中。

Data Time Shift: 该值的格式为 hh:mm:ss。安 EWC File Converter 可以将此值添加到 EWC 文件时间中。例如一个 EWC File Converter 可以创建一个从 00:00:00 开始的文件，并将下一个时间设置为 Data Time Shift 你进入这里。这两次的数据将是EWC 文件中的初始值。

Start and End Date: 该值的格式为 mmmdd。例如，将 1 月 1 日输入为 Jan01。安 EWC File Converter 可以使用它来输出运行瞬态模拟时使用的日期。

包含在 WPC 文件中

Wind Pressures: 选中此框以使转换器创建包含风压数据的 WPC 文件。

Contaminant Concentrations: 选中此框以使转换器创建包含污染物数据的 WPC 文件。在模拟过程中，WPC 文件中的污染物浓度优先于其他瞬态污染物设置。对于在 PRJ 文件中定义但未包含在 WPC 文件中的污染物，将使用稳态污染物数据，即默认浓度。

生成 WPC 文件： 使用此按钮激活用户开发的 WPC 创建程序。这将首先对 PRJ 文件执行检查并生成 PLD   如果所有信封开口均已正确定义，则归档。

生成 PLD 文件： 使用此按钮手动生成 PLD 文件。 PLD 文件将存储在与 PRJ 相同的位置   文件，与 PRJ 文件同名，但扩展名为 PLD。此文件对于使用外部编辑器创建 WPC 文件非常有用。
