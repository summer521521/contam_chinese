供应和返回点 - 明细表属性

时间表： 您可以将计划与送风/回风点相关联，以根据一天中的时间和一周中的某一天控制流入/流出送风/回风点所在房间的设计气流率的比例。

描述 ：如果由用户输入，则显示特定计划的更详细描述的字段。您可以通过按描述字段旁边的“编辑计划”按钮来修改此描述，然后更改计划的描述字段。

名称 ：此下拉框包含所有先前创建的计划的名称。从此框中选择的计划将定义正在修改的特定供应/返回系统的行为。

CVF/DVF 控制： 可以使用 CVF 或 DVF 文件中的值代替上述时间表（请阅读有关 CVF 和 DVF 控件的信息） 控制元件类型 部分）。这使得能够使用这些控制元素类型而无需绘制/定义关联的控制网络。关联的单选按钮“都不使用”、“使用CVF”和“使用DVF”用于选择所需的控制方法。从关联的“名称”下拉框中选择标题名称，该下拉框中将根据上述单选按钮选择显示 CVF 或 DVF 文件的标题。

控制状态： 这将显示当前的控制方法：计划、直接CVF、直接DVF或控制网络。如果 SketchPad 上控制网络的执行器与计划项目关联，则它将优先于计划和 CVF/DVF 控制。如果选择“使用 CVF”或“使用 DVF”单选按钮，则 CVF 或 DVF 将优先于关联的计划。
