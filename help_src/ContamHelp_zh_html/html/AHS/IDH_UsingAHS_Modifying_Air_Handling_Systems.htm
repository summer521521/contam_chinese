改造空气处理系统

要修改简单空气处理系统的参数，可以使用图标定义过程（请参见 定义建筑组件图标 ) 显示其属性并进行所需的更改。您可以移动空气处理系统图标以及移动和复制供气和回风图标；但是，您无法复制空气处理系统图标。

移动空气处理系统图标

您可以使用以下命令移动 AHS 图标 Cut and P 阿斯特 的职能 Edit 菜单。 ContamW 只允许您在其所在的同一级别内移动 AHS 图标。

移动供应和返回图标

定义供应或返回图标后，您可以使用 Cut and P 阿斯特 的职能 Edit 菜单。 ContamW 只允许您在当前所在的同一房间内移动供给或返回图标。

复制供应和返回图标

使用定义供应或返回图标后 Copy and P 阿斯特 的职能 Edit 菜单。您可以将供应和返回图标复制到任何级别的任何房间。
