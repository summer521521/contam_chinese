过滤器和时间表

枢纽站和航站楼的过滤器和时刻表有所不同。结点不能有与其关联的过滤器，并且只能计划或控制结点的温度。从 CONTAM 版本 2.4 开始，可以对终端进行过滤和调度。尽管一次只能应用一种类型的计划，但可以计划（或控制）终端温度和损耗系数。

过滤器： 您可以创建、编辑或删除与该终端关联的过滤器。创建新过滤器或修改现有过滤器将显示 过滤器属性 对话框。使用此对话框可以根据需要创建和编辑过滤器元素。有关污染物过滤器的说明，请参见 污染物过滤器 在 使用 CONTAM 手册的部分。

时间表： 从 CONTAM 版本 2.4 开始，端子在模拟过程中由管道段和端子接头组成。其中每一个都是可安排的项目。您可以选择安排 损耗系数 该段或 温度 交界处。明细表列表将包含项目中存在的明细表类型（无单位或温度）。

如果当前项目中有先前定义的计划，您可以从“名称”列表中选择一个。单击“新计划”按钮定义新计划。要查看或修改现有计划，请单击“编辑计划”按钮。将此字段设置为 < 无 > 表示没有时间表。 （参见 使用时间表 )

CVF/DVF 控制： 可以使用 CVF 或 DVF 文件中的值代替上述时间表（请阅读有关 CVF 和 DVF 控件的信息） 控制元件类型 部分）。这使得能够使用这些控制元素类型而无需绘制/定义关联的控制网络。关联的单选按钮“都不使用”、“使用CVF”和“使用DVF”用于选择所需的控制方法。从关联的“名称”下拉框中选择标题名称，该下拉框中将根据上述单选按钮选择显示 CVF 或 DVF 文件的标题。

控制状态： 这将显示当前的控制方法：计划、直接CVF、直接DVF或控制网络。如果 SketchPad 上控制网络的执行器与计划项目关联，则它将优先于计划和 CVF/DVF 控制。如果选择“使用 CVF”或“使用 DVF”单选按钮，则 CVF 或 DVF 将优先于关联的计划。
