逆止阻尼器：Q=C( D P)^n

这是逆止风门管道流量元件的体积流量版本。逆流阻尼器模型可以对具有不同流阻（取决于压降方向）的特征（例如防烟阻尼器）进行建模，并且在一个方向上的流量大大减少（或为零）。

姓名： 输入要用于标识风管流元素的唯一名称。该元素将保存在当前项目中，并且可以与多个风管段关联。

说明： 用于输入特定管道流元素的更详细描述的字段。

压差： 输入管道段正向和负向流向的流量系数和指数。 DP>0 指的是流元件两端的压差，该压差会导致正流方向上的气流（参见 风管段属性 ）和 DP<0 会导致相反方向的流动。

流量系数： 由于所使用的转换方法，系数只能以 SI 单位表示。使用以下转换将 IP 单位转换为 SI 单位。

流量指数： 流动指数变化范围为 0.5（流动受动态效应主导的大开口）和 1.0（流动受粘性效应主导的狭窄开口）。测量结果通常表明典型渗透开口的流量指数为 0.6 至 0.7。

形状尺寸和泄漏： 您必须输入数据来物理描述每个管道气流元件。您在 形状尺寸和泄漏量 与每个管道气流元件关联的属性页。
