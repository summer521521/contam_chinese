风机型号：定风量

该管道流量元素允许您将通过管道的气流描述为恒定的体积流量。您指定的设计流量是按标准空气计算的。 CONTAM 使用值 1.204 kg/m 3 基于 ASHRAE 规定的干燥空气条件：101.325 kPa 和 20°C [ASHRAE 第 18 章 2004年美国采暖及制冷工程师协会 ]。因此，如果模拟过程中的实际条件与这些标准条件不符，则由于空气密度的差异，结果将与指定流量不同。

姓名： 输入要用于标识风管流元素的唯一名称。该元素将保存在当前项目中，并且可以与多个风管段关联。

说明： 用于输入特定管道流元素的更详细描述的字段。

设计（最大）流量： 输入最大体积流量。您可以使用风管段明细表来修改该值。

形状尺寸和泄漏： 您必须输入数据来物理描述每个管道气流元件。您在 形状尺寸和泄漏量 与每个管道气流元件关联的属性页。
