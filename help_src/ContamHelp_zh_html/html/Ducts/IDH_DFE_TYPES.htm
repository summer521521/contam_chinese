管道流元件类型

这是可用管道流元素或数学模型的列表，提供管道段的气流和压差之间的关系。请参阅气流分析 理论背景 手册的数学细节部分。

达西-科尔布鲁克模型

CONTAM 管道系统最常用的模型是达西-韦斯巴赫关系和科尔布鲁克自然粗糙度函数。

达西-科尔布鲁克模型

幂律模型

包括幂律模型的三个版本：

  1. 孔口阻力模型

  2. 电阻：F = C( D P)^n （质量流量）

  3. 电阻：Q = C( D P)^n （体积流量）

强制流模型

包括三种类型的强制流动模型：

  1. 风扇-性能曲线

  2. 恒定质量流量

  3. 定体积流量

逆止阻尼器模型

逆流阻尼器模型可以对具有不同流阻（取决于压降方向）的特征（例如防烟阻尼器）进行建模，并且在一个方向上的流量大大减少（或为零）。

  1. 逆止阻尼器：F = C( D P)^n （质量流量）

  2. 逆止阻尼器：Q = C(D P)^n （体积流量）

三次样条流模型

这些模型能够根据三次样条曲线拟合的一组数据点创建用户定义的管道流元素。有四种版本将质量流量或体积流量与流量元件上的压差相关（反之亦然）。三次样条曲线拟合确保流量和压力之间的关系按照 CONTAM 气流求解器的要求是可微分的。

  1. F 与 P : 质量流量 作为一个函数 压差

  2. Q 与 P : 体积流量 作为一个函数 压差

  3. P 与 F : 压差 作为一个函数 质量流量

  4. P 与 Q : 压差 作为一个函数 体积流量
