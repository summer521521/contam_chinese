删除管道

删除当前定义的管道是一个两步过程。也就是说，如果要删除风管段、交汇点或终端，必须首先取消定义该项目。 取消定义风管段 您必须突出显示特殊管道段图标（指示正流向）并按 删除 键或从编辑菜单中选择删除。这将从两个端点突出显示整个风管段，并通过显示消息框提示您确认取消定义操作。一旦取消定义风管段，它将再次显示为红色，并且特殊风管段图标将转换回直线段图标。 删除风管段 您突出显示未定义段或关联的未定义端子的任何部分并执行删除。要删除的整个片段将突出显示，并且系统会要求您确认删除。

删除管道时应考虑几个事项。您无法删除仍连接到已定义风管段的未定义交汇点。如果删除两个未定义线段之间的交汇点，该交汇点将被删除以形成单个线段。您无法在不删除其关联段的情况下删除终端。

删除 SketchPad 上定义的管道不会删除您可能已定义的任何管道流元素。这些流程元素只能使用 CONTAM 库管理器删除，可通过 Data 每种类型的数据元素的菜单。
