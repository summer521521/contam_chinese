三次样条模型

三次样条模型使您能够根据与用户定义的数据点集的曲线拟合来创建气流元件。用于生成曲线的三次样条拟合保证了 CONTAM 求解器所需的流量和压力之间的一阶可微关系。在模拟过程中，压力差的符号将基于 P 积极的 Flow D 指示 为与样条线元素关联的流动路径定义。例如，正流方向上的压力下降（即 来自房间 – 压力 到房间 > 0) 将利用样条数据的正压力。

每个元素都有一些基本要求。它们至少需要四个数据点，因此在输入最小数量的点之前不会显示曲线。所有模型都要求曲线拟合的所有段的斜率都大于零。如果单击“确定”按钮时曲线拟合出现错误，将显示一条错误消息，指示曲线的违规部分以及错误原因。段的编号从零到数据点的数量减一。例如——“ 段 0：y ’ <= 0 如果前两个数据点之间的线段的斜率小于或等于零，将显示“。

姓名： 输入要用于识别气流元件的名称。

说明： 用于输入特定气流元件的更详细描述的字段。

曲线数据： 创建和编辑数据点列表以定义气流元件的曲线。您正在编辑的三次样条元素的类型决定了曲线的自变量和因变量。列表的标签、数据输入字段和关联的单位将相应更改，绘图的轴也会相应更改。使用数据输入字段和“添加”按钮来创建新数据点和编辑现有数据点。使用“删除”按钮从列表中删除当前选定的数据点。

图标： 根据特定气流元件选择小或大开口图标。该图标对模拟没有影响。

质量流量与元件压降的函数关系。

三次样条：Q 与 P

体积流量与元件压降的函数关系。

三次样条：P 与 F

压降是通过元件的质量流量的函数。

三次样条：P 与 Q

压降是通过元件的体积流量的函数。
