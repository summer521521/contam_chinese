气流元件类型

这是可用气流元件或数学模型的列表，提供气流路径的气流和压差之间的关系。请参阅气流分析 理论背景 手册的数学细节部分。

使用幂律模型的单向流

这些单向流动模型允许沿压降方向流动。 CONTAM 为您提供以下幂律流动模型。

  1. Q = C( D P)^n ：这是体积流量形式的幂律模型的一般形式，允许您直接输入系数， C ，和指数， n.

  2. F = C( D P)^n ：这是质量流量形式的幂律模型的一般形式，允许您直接输入系数， C ，和指数， n.

  3. 泄漏面积数据 ：该模型是指有效泄漏面积，如《规范》第 26 章中所述。 2001 ASHRAE 基础知识手册 .

  4. 连接 (ASCOS) 数据 ：指ASCOS程序中使用的气流描述[ 克洛特 1982 ].

  5. 孔口面积数据 ：将开口描述与孔口面积数据相关联。

  6. 裂纹描述 ：通过其长度和宽度描述的狭窄开口。

  7. 测试数据（1分） ：使用单一流量和压降以及压力指数的估计值， n.

  8. 测试数据（2分） ：使用两个流速及其相应的压降来使用幂律模型定义流量。

  9. 楼梯间 ：楼梯间对应的数据符合幂律模型。

  10. 轴 ：描述大竖井中的流动。

使用二次模型的单向流

这些模型仅允许基于压降沿一个方向流动通过路径。

  1. D P = aQ + bQ^2 and D P = aF + bF^2 ：这些模型允许您直接输入二次模型的“a”和“b”系数。

  2. 裂纹描述 ：该模型比幂律版本的裂纹模型更详细地描述了狭窄的开口。

  3. 测试数据（2点） ：该模型使用两个流速及其相应的压降来使用二次模型定义流量。

双向流模型

这些模型用于表示高开口，开口处的温差可能会导致开口处的双向流动，例如门口。

  1. 具有 2 路流动的单开口型号 ：这会将通过路径的流量建模为单个大开口。

  2. 两开型号 ：该模型垂直划分一个开口，并使用两个幂律模型（每个开口一个）对流量进行建模。结果以每个方向的净流量的形式获得。

逆流阻尼器流动模型

这些模型允许您根据压降方向对具有不同阻力的特征进行建模，例如排烟风门。 CONTAM 实现了反向气流阻尼器模型的体积流量和质量流量版本，这两个版本都基于幂律模型。

  1. 逆止阻尼器模型：F = C( D P)^n ：这是逆风阻尼器模型的体积流量形式。您可以根据阻尼器压降的符号输入单独的 C 和 n 值。

  2. 逆止阻尼器模型：Q = C(dP)^n ：这是逆风阻尼器模型的质量流量形式。您可以根据阻尼器压降的符号输入单独的 C 和 n 值。

  3. 自调节通风口 ：该元件限制通过具有用户定义的限制气压差的流路的两个方向的气流速率。 CONTAM 实现自调节通风口，如 [ 阿克斯利 2001 ].

风扇和强制流动模型

这些模型允许您创建代表风扇的气流路径或您要为其指定流速的其他路径。

  1. 恒质量流量风机 ：该模型提供恒定的质量流量。

  2. 定风量风机 ：该模型提供恒定的体积流量，但质量流量将根据实际密度而变化。

  3. 三次多项式风扇模型 ：该模型实现了用户定义的风扇性能曲线来关联压力与流量。

三次样条流模型

这些模型能够根据三次样条曲线拟合的一组数据点创建用户定义的气流元件。有四种版本将质量流量或体积流量与流量元件上的压差相关（反之亦然）。三次样条曲线拟合确保流量和压力之间的关系按照 CONTAM 气流求解器的要求是可微分的。

  1. F 与 P : 质量流量 作为一个函数 压差

  2. Q 与 P : 体积 流量 作为一个函数 压差

  3. P 与 F : 压差 作为一个函数 质量流量

  4. P 与 Q : 压差 作为一个函数 体积 流量
