二次模型： D P = aQ + bQ²

该模型允许您直接将系数 a 和 b 输入到二次流量模型的体积流量版本中。在哪里 D P 是压降，Q 是体积流量。

姓名： 输入要用于识别气流元件的名称。气流元件将保存在当前项目中，并且可以与多个气流路径关联。

系数： 由于 ContamW 处理单位转换的方式，系数必须以 SI 单位表示。标准条件下的系数单位，即 sm 3 /s，如下：

一个[帕 • 不锈钢/不锈钢 3]

b [帕 • （不锈钢/不锈钢 3)2]

将体积流量 Q 换算为 m 单位 3 /s 密度 r 发给短信 3 /s 密度 rs ，将 Q 乘以 r/rs , 其中 rs 等于 1.2041 千克/米 3 在20°C。

说明： 用于输入特定气流元件的更详细描述的字段。

图标： 根据特定气流元件选择小或大开口图标。该图标对模拟没有影响。
