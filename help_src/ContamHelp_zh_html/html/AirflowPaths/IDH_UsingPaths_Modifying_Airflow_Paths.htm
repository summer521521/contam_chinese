修改气流路径

要修改气流路径的参数，可以使用图标定义过程（请参见 定义建筑组件图标 ) 显示其属性并进行所需的更改。您还可以移动和复制气流路径。

移动气流路径图标

定义气流路径后，您可以使用 Cut and P 阿斯特 的职能 Edit 菜单。 ContamW 将根据以下规则约束路径图标的移动。您只能将流路穿过墙壁移动到墙壁上，而不能移动到房间的地板上。您只能将流路穿过地板移动到地板上的另一个位置，而不能移动到墙壁上。您只能在其所在的同一级别内移动路径图标。

复制气流路径图标

定义气流路径后，您可以使用复制路径图标 Copy and P 阿斯特 的职能 Edit 菜单。 ContamW 将根据以下规则限制路径图标的复制。您只能将穿过墙壁的流路复制到墙壁上，而不能复制到房间的地板上。您只能将穿过地板的流路复制到地板上的另一个位置，而不能复制到墙上。您可以将路径图标从一个级别复制到另一级别。
