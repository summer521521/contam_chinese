使用项目注释

项目 注释 和 项目描述 为您提供简单的项目文档功能。

注释

您只需将注释图标放置到 SketchPad 上（请参阅 绘制建筑组件图标 ），然后使用图标定义过程（参见 定义建筑组件图标 ) 编辑注释文本。一旦定义了与注释图标关联的文本，每当您用插入符号突出显示该图标时，该文本就会出现在状态栏中。您可以像编辑任何其他建筑组件图标一样编辑、删除和移动该图标。

颜色: 设置图标的颜色。这允许您区分 SketchPad 上的不同图标。您需要从“视图”菜单中设置“图标颜色”以显示非默认颜色。

项目描述

提供项目文件描述以帮助您管理项目文件。当您从 ContamW 中打开项目文件时，此描述将可用（请参阅 使用项目文件 ).
