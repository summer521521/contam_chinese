运行 CONTAM

如中所示 安装 CONTAM 部分，CONTAM 实际上由两个可执行程序组成：ContamW 和 ContamX。 ContamW 是图形界面，ContamX 是模拟引擎。通常，您将通过 ContamW 中的菜单命令激活 ContamX。但是，ContamX 可以用作命令行工具，甚至可以在脚本或批处理文件中使用。

运行 ContamW

通过选择运行 ContamW CONTAM #.#.#.# （其中 # 表示程序版本，例如 3.4.0.0）从“开始”菜单。

作为命令行程序运行 ContamX

将 ContamX 作为命令行程序运行的最简单方法是将 contamx3.exe 文件以及相关项目和支持文件复制到公共目录。可以通过键入以下内容来显示运行 ContamX 的选项：

contamx3 --帮助

这将显示以下输出：

ContamX - 多房间气流和污染物传输分析软件

用法：contam-x [输入文件] [选项]

-h, --help 显示此帮助消息并退出

-v, --Version 显示该程序的版本并退出

-b，--桥= <str> 在桥接模式下运行 ContamX，可选 < IP地址:端口 >

-w, --Wind ContamX 根据风速计算风压

       通过 ADJ_WTH_MSGTYPE 获得的方向

-f, --volumeFlow ContamX 通过桥输出体积气流速率，单位为 [m3/s]

-c, --Console 使用交互式控制台运行 ContamX

-d, --Dialog 使用 Windows 对话框运行 ContamX

-t, --TestInput 仅测试输入

[input-file] 是 CONTAM PRJ 文件的路径。

     如果省略且没有其他选项，系统将提示您输入文件名。

     可以通过拖动在 Windows 资源管理器中指定该文件

     将项目文件图标拖到 ContamX 程序图标上。

使用 TCP/IP 套接字通信运行 ContamX

ContamX 可以在 TCP/IP 套接字通信模式下运行，该模式可以对仿真进行运行时控制，包括通过 TCP/IP 套接字执行控制和数据传输。 参见 附录B 了解 TCP/IP 消息结构的详细信息和完整描述。
