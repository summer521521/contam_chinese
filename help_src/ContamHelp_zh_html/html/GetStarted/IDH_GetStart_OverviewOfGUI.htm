CONTAM 图形界面

ContamW 是 CONTAM 的图形界面 (GUI)。使用 ContamW 创建和查看您的气流与污染物扩散分析项目。它主要由一个称为 SketchPad 的绘图区、一组绘图工具、一个标题栏、一组菜单和一个状态栏组成。以下部分提供了 CONTAM GUI 的每个功能的简要说明。请参阅 使用 ContamW 部分了解如何使用这些功能的详细信息。

 

图 - CONTAM 图形界面

标题栏

标题栏是 ContamW 主窗口顶部的典型矩形区域。 CONTAM 项目文件名将显示在此区域中。

菜单

该菜单是典型的 Windows 程序，但具有提供特定于 CONTAM 应用程序的功能的差异。通过此菜单，可以执行大多数 ContamW 操作，包括：保存和检索项目文件、选择各种显示模式、设置和执行模拟以及访问在线帮助系统。请注意，某些菜单项具有可快速访问的快捷键或热键；例如，要保存当前项目文件，请使用 Ctrl+S 组合键。

工具栏

工具栏（如下图所示）出现在菜单下方，并提供某些菜单项的便捷快捷方式。一些工具栏按钮与其他 Windows 应用程序中的按钮类似。其他按钮提供了特定于 ContamW 的功能的快捷方式。

状态栏

如下图所示，状态栏是主窗口底部、SketchPad 下方显示的区域。该区域分为三个独立的窗格，根据 SketchPad 的当前模式显示各种信息。

ContamW 2.4 现在为您提供了一个浮动状态栏，每当您在屏幕上选择某些图标时，该状态栏就会显示在光标附近，从而提供更方便的方式来查看图标信息。这类似于当您将 Windows 光标悬停在工具栏按钮上时出现的工具提示。您可以通过以下方式打开和关闭此功能 View → Floating Status Bar or Ctrl+T 获取工具提示。

左窗格

该窗格始终显示建筑组件图标的类型和数量，例如房间、路径、空气处理系统等。

在 常规模式 操作时，最左边的窗格显示当前突出显示的单元格或图标的摘要信息。

在模拟中 结果模式 ，最左边的窗格显示当前突出显示的图标的结果。对于房间，这包括相对于该区域所在水平面高度处的环境压力的温度和压力。对于管道接头和终端，它显示相对于接头或终端所在房间的温度和静压。对于路径和管道段，它将显示路径或段上的气流和压降以及指示流动方向的符号（ >, < , ^, v)。对于简单的空气处理系统图标，将显示隐式流动路径的室外气流、再循环气流和排气气流。

许多显示单位可以通过在项目配置属性中选择单位的 SI 或 IP 系统来控制，该属性可通过 View →Options... 菜单命令。

中心窗格

在 常规模式 在操作过程中，中心窗格指示当前突出显示的图标或单元格在 SketchPad 坐标中的位置（从左上角开始编号）。在模拟中 结果模式 中心窗格显示当前模拟时间步长，并显示其结果。

右窗格

最右侧的窗格显示当前级别的名称和编号以及项目中的级别总数。
