开始使用

安装 CONTAM

获取 CONTAM

CONTAM 从一组安装文件进行安装，您可以从 NIST 获取这些安装文件。这些文件可以从 NIST 网站下载（ https://www.nist.gov/el/energy-and-environment-division-73200/nist-multizone-modeling ).

安装 CONTAM

从 NIST 网站下载 CONTAM 后，双击安装程序可执行文件 (.exe) 开始安装过程。选择要安装程序的文件夹，或者只需选择默认位置“C:\Program Files (x86)\NIST\CONTAM XX”（其中 XX 指版本号）。按照说明完成安装。此安装不会删除或覆盖以前版本的 CONTAM。

安装包括以下项目：

CONTAM：

contamw3.exe – CONTAM 图形界面，ContamW。

contamx3.exe – CONTAM 模拟引擎 ContamX。

ContamHelp.chm – CONTAM 用户指南和上下文相关帮助可通过 ContamW 访问。

olch2d32.dll – 用于 ContamW 图表显示的动态链接库。

实用程序：这些是由命令行界面组成的控制台程序，在下面的运行实用程序部分中对此进行了解释。

simread.exe – 读取二进制 CONTAM 模拟结果文件（SIM 文件）并将数据输出到文本文件。该实用程序可用于自动提取和转换模拟结果以进行定制后处理，例如通过电子表格或 Python 脚本。

prjup.exe – 升级 CONTAM PRJ 文件。 ContamW 还会将以前版本的 PRJ 文件升级到当前版本。该程序是为了支持 CONTAM 开发人员的测试而开发的。

simcomp.exe – 比较两个模拟结果 SIM 文件。该程序是由 CONTAM 的开发人员开发来支持回归测试的。当用户转换到较新版本的 CONTAM 时，它还可以帮助用户评估不同版本执行的计算之间的潜在差异。

正在卸载 CONTAM

CONTAM 安装程序还为您提供卸载功能。您可以像卸载典型的 Windows 程序一样卸载 CONTAM。
