修改本地数据

您可以使用 CONTAMW 库管理器修改数据元素的本地列表。库管理器允许您创建新的本地数据元素、将数据元素从库复制到本地项目、修改本地数据元素以及删除本地数据元素。

创建本地数据元素

您可以使用库管理器的“项目元素”部分中包含的“新建”按钮来创建新的数据元素。您只能创建库管理器当前显示的数据元素类型（例如气流元件、管道流元素、明细表等）。 “新建”按钮将激活与创建当前正在使用的数据元素类型相关的 ContamW 对话框。

必须有可用的物种才能创建新的污染物相关元素，例如源/汇、过滤器和动力学反应。

复制本地数据元素（在项目和库之间）

See 复制库数据元素 .

修改本地数据元素

库管理器为您提供了另一种访问本地项目元素的方法。这是对本手册的使用 CONTAM 部分中介绍的访问建筑组件属性的方法的补充。您必须首先在库管理器的项目元素部分的元素列表中突出显示元素名称。然后单击列表旁边的“编辑”按钮。这将激活与显示当前正在使用的数据元素类型的属性相关的 ContamW 对话框。然后进行所需的更改并单击“确定”按钮以使更改生效。

复制数据元素（在项目内）

您可以使用库管理器创建与现有元素相似但名称不同的元素。为此，您将项目元素复制到库中（不必打开现有的库文件），为库元素指定一个新名称，进行所需的修改，然后将该元素复制回项目中。

删除本地数据元素

仅当项目中的任何建筑组件未引用本地数据元素时，您才可以删除它们。要删除项目数据元素，请在 CONTAMW 库管理器的项目元素部分中突出显示元素名称，然后单击列表旁边的“删除”按钮。仅当项目中未引用当前突出显示的元素时，才会启用“删除”按钮。系统将提示您确认删除，并且该数据元素将不再显示在项目元素列表中。但是，在保存项目文件之前，删除不会受到影响。只要删除元素后不保存文件，您就可以通过重新打开项目文件来检索以前删除的数据元素。

删除具有关联子元素的数据元素不会删除子元素。您必须使用单独的库管理器操作删除子元素。
