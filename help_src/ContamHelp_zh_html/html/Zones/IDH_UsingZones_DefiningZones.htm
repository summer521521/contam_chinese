定义房间

绘制房间的封闭房间后，必须定义该区域。要定义房间，您必须首先在封闭房间内绘制房间图标（请参见 绘制建筑组件图标 ），然后使用图标定义过程（参见 定义建筑组件图标 ) 显示和编辑房间的属性。

每个房间的属性包括名称、温度（恒定或预定）、压力（变量或恒定）、体积、描述房间内污染物行为的信息以及一维房间信息。您必须为每个房间提供一个对于项目的每个级别而言唯一的名称。因此，您无法复制关卡内的房间数据。但是，您可以将整个级别的数据（包括房间数据）从一个级别复制到另一个级别（请参阅 创建关卡 ）。房间属性的详细描述如下 房间属性 下面。
