单元格/图标大小

当前单元格/图标大小： 在 SketchPad 上显示图标有七个选项（1x1、2x2、3x3、4x4、5x5、8x8 和 16x16）。默认选项是 8x8。这是一种“缩放”功能，可提供视觉帮助，使您能够更详细地查看草图或在程序窗口中显示更大的草图。单元大小对项目模拟没有影响。工具栏上提供了两个“缩放”按钮，可根据下面可用的单元格大小快速增大和减小单元格大小。

可用的单元尺寸： 使用这些复选框来选择您想要的可用尺寸。 1x1 大小会导致某些版本的 Window 98 操作系统出现问题。如果您遇到这种情况，请取消选择 1x1 复选框并单击“保存配置”按钮。

单元格跳转： 使用此选项可设置当您使用 Shift + 箭头键在 SketchPad 周围进行操作时光标将跳转的 SketchPad 单元格数量。
