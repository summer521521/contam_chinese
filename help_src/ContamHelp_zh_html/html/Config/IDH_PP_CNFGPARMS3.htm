SketchPad 外观

如果保存配置文件，则每当您启动 ContamW 时，这些选项将默认为该属性页上设置的值。

SketchPad 选项

浮动状态： 选中此框可在 SketchPad 上显示浮动状态框。您还可以通过以下方式打开和关闭浮动状态框 View → 弗洛亚 t 状态 (Ctrl+T) 菜单命令。

颜色房间： 选中此框可在 SketchPad 上显示房间填充颜色。如果未选中此框，则将使用“背景”调色板颜色填充所有房间。您还可以通过以下方式打开和关闭颜色房间选项 View → 颜色 Z 个 (Ctrl+Z) 菜单命令。

彩色导管： 选中此框可在 SketchPad 上显示管道颜色。如果未选中此框，则所有管道将使用“管道”调色板颜色显示。您还可以通过以下方式打开和关闭颜色管道选项 View → 颜色 Z 个 (Ctrl+Z) 菜单命令。

调色板： ContamW 提供修改 SketchPad 上显示的项目颜色的功能。您可以选择四个可用调色板之一。您无法修改“默认”调色板的颜色，但可以修改其他三个“用户调色板”。选择一个调色板以在调色板中显示该调色板的颜色 调色板颜色 下面的部分。

调色板颜色： 这些是用于在 SketchPad 上显示项目的颜色。单击颜色框可更改用于显示每种类型的项目的颜色。只能更改用户调色板的颜色。以下是每个类别的简要说明。

  • 前景 - 定义的图标，包括墙壁

  • 背景 - SketchPad 单元格的背景、默认房间填充和所有房间的填充 色区 选项未激活。

  • 流量结果 - 显示气流路径和管道终端的缩放流量结果的线

  • 压力结果 - 在结果和风压视图模式下显示气流路径上的缩放压差结果的线

  • 绘图线 - 在 SketchPad 上绘图时单元格突出显示

  • 未定义 - 未定义的图标

  • 控件 - 定义的控件图标

  • 管道 - 定义的管道的默认颜色以及所有定义的管道的颜色 彩色导管 查看选项未激活

  • 错误 - 当错误条件在 SketchPad 上突出显示时单元格的背景颜色。当建筑物检查失败时会指示错误。当您选择时会执行建筑物检查 R 联合国模拟 or 创建CONTAM X 输入文件 从 S 模拟 菜单。

  • 光标 - SketchPad 绘图和图标选择光标

  • 突出显示 - 用于查找操作的图标突出显示颜色

  • 子级别 - 通过激活子级别显示选项时子级别上的所有图标 L 埃维尔 → 显示等级 B 低 (Ctrl+Shift+B) 菜单命令。
