创建关卡

当您第一次启动 ContamW 时，会创建一个默认级别，以便您可以立即开始处理绘图，而无需创建新级别。里面有3个命令 L 埃维尔 创建新关卡时使用的菜单。这些是 C 操作级别 , P 品味等级 , 和 I 插入空白级别 命令。每当您创建新级别时，无论它是空白的还是其他级别的副本，ContamW 都会为其指定一个默认名称，该名称由包含在“ < ” 和 “ > " 字符。您可以稍后通过编辑与级别关联的数据来修改此名称。

在 ContamW 中复制级别时必须小心。例如，您可能在某个层的管道系统中定义了空气处理器，然后将其复制到另一个层。这将在新的水平上创建另一个空气处理器。如果管道系统连接在两层之间，则两个空气处理器可能会相互影响。您必须小心地以对您的目的有意义的方式在建筑物楼层之间建立连接。

创建空白级别

菜单命令：  L 埃维尔 → I 插入空白级别 (A 博韦 /B 低电流水平 )
键盘快捷键：   无
工具栏按钮：   无

使用这些命令创建空白关卡。创建空白层时，必须选择是要在当前层之上还是之下创建它。

复制和粘贴关卡

菜单命令：  L 埃维尔 → C 操作级别 连同
   L 埃维尔 → P 品味等级 (A 博韦 /B 低电流水平 )
键盘快捷键：   无
工具栏按钮：   无

使用 ContamW 您可以复制整个关卡并将其作为整个关卡插入当前项目中的任何位置。使用复制级别命令复制SketchPad上当前显示的级别；移动到要插入复制的关卡的上方或下方的关卡，然后使用“粘贴关卡”命令之一插入复制的关卡。
