CONTAM 的特殊应用

本节包括使用 CONTAM 模拟多种不同建筑特征（例如中庭、楼梯间和烟囱）并执行特殊建筑分析（例如风扇增压）的说明或注意事项。还有一些可用的参考资料介绍了 CONTAM 在各种研究中的应用[ 艾默里奇与珀西里 1996 and 1998; 艾默里奇和纳宾格 2000 ; 方和珀西里 1995 ; 马瑟和尤尔 1999 ; 穆瑟2000 ; 波斯利 1998 ; 佩尔西里和马丁 2000 ; 珀斯利与常春藤 2001 ]。另外，请检查 NIST IAQ Analysis 网站上提供的参考书目 http://www.bfrl.nist.gov/IAQanalysis/pubs.htm 。有些文件可直接从网站获取。
