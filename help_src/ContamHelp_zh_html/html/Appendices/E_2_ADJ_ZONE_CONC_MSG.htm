ADJ_ZONE_CONC_MSGTYPE

ContamX 可能会在任何时间收到此消息 就绪模式 。污染物浓度或质量应按列表中指定的房间顺序发送 房间ID 。每条消息可用于设置多个房间的浓度/质量，但在处理多个代理时，必须为每个代理发送单独的消息。这里使用的房间和代理 ID 是由 AGENT_INFO_MSGTYPE and ZONE_INFO_MSGTYPE 分别。

消息头；

int 选项；    // 0 = 添加质量 [kg]，1 = 设置浓度 [kg/kg]

int num_zones； //要设置的房间数量

int 代理 ID；   // 污染物指数  

int zone_ids[num_zones]; // 房间索引列表

浮动 值[num_zones]； // 质量或浓度列表
