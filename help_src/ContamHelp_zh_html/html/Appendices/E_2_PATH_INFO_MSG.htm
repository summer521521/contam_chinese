PATH_INFO_MSGTYPE

提供与“标准”气流路径（相对于 AHS 路径）相关的信息。 的 编号[] 数组提供 ContamX 通过以下方式提供气流速率的顺序 PATH_FLOW_UPDATE_MSGTYPE 。如果 环境索引 为 0 时，该路径不是包络渗透点。 否则， 环境索引 指定使用以下命令发送路径浓度的顺序 ADJ_WPC_MSGTYPE 。正流向基于来自房间的气流路径数据 n 到房间 m.

消息头；

int 路径数； // PRJ 中的气流路径数

int id[路径数]; // 根据 ContamX 的索引

int 环境索引[路径数] // 0 = 未连接到环境

int element_index[num_paths]; // 初始气流元件索引

int from_zone[num_paths]; // from 房间的房间编号（基于正流向）

int to_zone[num_paths];    // 目标房间的房间编号

浮动 x[路径数]; // x坐标[米]

浮动 y[路径数]; // y 坐标[米]

浮动 z[路径数]； // 如果环境索引 > 0 则为绝对高度，否则相对于水平面 [m]
