ADJ_ZONE_HR_MSGTYPE

ContamX 可能会在任何时间收到此消息 就绪模式 。每条消息可用于设置多个房间的湿度比。这里使用的 ID 是由 ZONE_INFO_MSGTYPE .

注意：目前这仅适用于 CONTAM 与 EnergyPlus 耦合时使用。当将质量气流速率转换为 PATH_FLOW_UPDATE_MSG 的体积流量时，ContamX 将使用这些值来计算房间密度；然而，它并没有利用这些值来计算污染物迁移。

消息头；

int num_zones；                 //要设置的房间数量

int id[num_zones];       // 房间索引列表

浮动 值[num_zones]； // 湿度比[kg/kg]
