CX_ADVANCE_MSGTYPE

ContamX 在任何时间都会收到此消息 就绪模式 通知 ContamX 运行，直到从模拟第一天的时间 0:00:00 开始指定的以秒为单位的模拟时间（由 PRJ 文件中的模拟开始时间定义）。例如，如果模拟从 09:30:00 开始，模拟时间步长为 5 分钟，则第一个 时间 发送的时间应至少为 34200 秒 + 1800 秒。的 选项 字段用于指示 ContamX 应以以下形式发回的数据 更新消息 达到指定时间后。

消息头；

int 选项； // 按位标志控制 ContamX 将发送哪个 _UPDATE_MSGTYPES

            // 一次 时间 已达到

  • bit 1：迭代提前消息（当前未使用）
  • 位 2：发送 CONC_UPDATE_MSGTYPE
  • 位 3：发送 PATH_UPDATE_MSGTYPE
  • 位 4：发送 AHSP_UPDATE_MSGTYPE
  • 位 5：发送 DUCT_UPDATE_MSGTYPE
  • 位 6：发送 TERM_UPDATE_MSGTYPE
  • 位 7：发送 LEAK_UPDATE_MSGTYPE
  • 位 8：发送 CTRL_NODE_UPDATE_MSGTYPE

int 时间； // ContamX 应提前的模拟时间（以秒为单位）
