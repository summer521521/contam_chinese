ADJ_WPC_MSGTYPE

ContamX 可以随时在 就绪模式 。它可用于设置包络气流路径处的环境浓度或压力，包括简单空气处理系统和管道终端的室外进气口。路径、简易空气处理系统室外进风口和末端的浓度按照环境指标规定的顺序传输 PATH_INFO_MSGTYPE , AHSP_INFO_MSGTYPE , 和 TERM_INFO_MSGTYPE 消息。如果需要传输多个agent，顺序为：

  [路径 1 的所有代理]，

...

  [路径 num_paths 的所有代理]，

  [ahs 路径 1 的所有代理]，

 ...

  [路径 num_ahsp 的所有代理]，

  [1号航站楼的所有代理]，

 ...

  [终端 num_terms 的所有代理]

 

If 代理数量 为 0，则消息应提供与包络位置处的压力相对应的一组压力，而不是浓度。压力必须符合 WPC 文件的 2.4 格式，如 WPC 文件格式 文档部分。

消息头；

int 时间； //（需要但当前未使用的值）

int 代理数量； // 污染物数量或设置为零以发送压力

int num_vals； // 根据 num_agents 参数发送的项目数

对于污染物，即 num_agents > 0:

int agent_ids[num_agents]; // 正在发送值的每种污染物的索引

浮动 值[num_agents * nEnvPaths]； // 浓度 [kg_cont/kg_air]

对于风压，即 num_agents == 0：

浮动 值[nEnvPaths]； // 风压 [Pa]
