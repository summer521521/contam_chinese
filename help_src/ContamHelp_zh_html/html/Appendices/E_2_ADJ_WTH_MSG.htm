ADJ_WTH_MSGTYPE

ContamX 可能会在任何时间收到此消息 就绪模式 。每条消息均可用于设置天气条件，包括温度、气压、风速和风向，类似于 WTH 的方式   使用 WPC 文件而不是通过 WPC 文件获得详细的风压信息，并且 ADJ_WPC_MSGTYPE 。请参阅 程序执行 部分了解与使用此消息相关的信息。

从 CONTAM 3.3 开始，此消息包括环境湿度比。

消息头；

浮动 温度；      // 环境干球温度 [K]

浮动 压力；        // PRJ 参考高度处的大气压 [Pa]

浮动 风速；      // 风速[米/秒]

浮动 风向；   // 风吹的方向[度]

浮动 湿度比 ;  // 湿度比[kg/kg]
