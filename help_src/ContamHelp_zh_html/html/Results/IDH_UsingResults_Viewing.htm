查看结果

本节介绍如何激活 ContamW 的结果显示模式，并在瞬态仿真的 SketchPad 上显示不同时间步的结果。

结果显示方式

要使用 ContamW 查看结果，您可以通过选中以下任一选项来激活结果显示模式： S ketchPad 结果 或 R 结果显示窗口 选择的 View 菜单。这将在成功完成模拟后或打开具有与其关联的模拟结果文件的项目文件后自动发生。一旦激活结果显示模式，状态栏将显示结果相关信息（参见 状态栏 ），您将能够利用所有 结果显示方法 概述如下。

时间步长

如果您执行了瞬态仿真，则每个时间步都将提供结果。但是，一次只能显示一个时间步长的 SketchPad 结果（如下节所述）。您可以使用以下菜单和键盘快捷键命令更改当前在 SketchPad 上显示结果的时间步长 View → 结果 N 航空 菜单。

显示下一个时间步
菜单命令：  N 下一个时间步长
键盘快捷键：  End

显示上一个时间步
菜单命令：  P 前一个时间步长
键盘快捷键：   首页

显示最后一个时间步（当天）
菜单命令：  L 时间步长
键盘快捷键：   Ctrl+结束

显示第一个时间步（当天）
菜单命令：  F 第一个时间步长
键盘快捷键：   Ctrl+Home

可以通过项目配置的结果显示属性设置以下“跳转”命令的增量。

向前跳转（n 个时间步）
菜单命令：   跳跃 A 头
键盘快捷键：   Shift+结束

向后跳（n 个时间步）
菜单命令：   跳跃 Back
键盘快捷键：   Shift+Home

转到日期/时间...
菜单命令：  G o 日期/时间
键盘快捷键：   Ctrl+G
