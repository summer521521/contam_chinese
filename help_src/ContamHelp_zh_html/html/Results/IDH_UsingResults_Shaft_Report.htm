轴报告

CONTAM 非常适合烟雾控制系统的分析。由于竖井对于某些排烟系统的重要性[ 克洛特和米尔克 2002 ], ContamW 为轴提供特殊的报告功能。

定义轴

您可以在项目开发的绘图阶段定义竖井，方法是将房间和气流路径图标直接放置在建筑物相邻楼层的彼此下方（同一列和行中）。如果在当前层上方或下方的层的相同位置上存在房间和路径图标，则在 ContamW 生成井道报告时将利用这些房间和路径。

您可以通过选择创建轴报告 生成 S 半报告 从 S 模拟 菜单。这将显示一个对话框，提示您执行以下三个步骤来定义与您希望 ContamW 报告气流和压降的轴相关的轴房间图标以及主要和次要流动路径。房间图标通常代表楼梯间，主要流动路径通常代表楼梯间门。

为轴报告选择轴的步骤：
1) 左键单击定义轴的房间图标
2) 左键单击主气流路径图标
3) 左键单击辅助气流路径图标
注：单击鼠标右键可取消轴报告生成过程。

完成上述步骤后，轴报告将显示在对话框中。该报告将显示轴每一层的压降、气流速率和气流方向，其中 SketchPad 上与主要和次要气流路径相同的位置有气流路径图标。您可以使用“轴报告”对话框选择报告气流速率和压降的单位，并且可以将报告的副本保存到文本文件中。
