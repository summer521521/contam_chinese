打印图表

图形格式： 有四种图形格式可用于打印图表。

位图：这会生成一个标准的 Windows BMP，其颜色深度与当前用户的视频模式相同。

增强型图元文件：一个简单的增强型 Windows 图元文件。

Aldus 可放置图元文件：这种图元文件在放入文档时往往比其他图元文件效果更好。

Windows 图元文件：标准 Windows 图元文件。

规模： 打印 ContamW 图表时有 5 种不同的缩放选项可用。

无缩放：图像按原样打印，与纸张尺寸无关。

缩放到宽度：缩放图表的宽度以匹配打印输出的宽度，忽略高度。

缩放到高度：缩放图表的高度以匹配打印输出的高度，忽略宽度。

缩放以适合（保持宽高比）：缩放打印输出以尽可能匹配打印输出的尺寸，同时保持原始宽高比。

缩放以适合（忽略宽高比）：缩放打印输出以尽可能匹配打印输出的尺寸，而不考虑原始宽高比。此选项将使图表尽可能接近页面，但可能会稍微拉伸/扭曲图表。

复制到剪贴板： 此选项允许您以上述指定的格式将图表数据复制到剪贴板，以便在其他 Windows 应用程序中使用。
