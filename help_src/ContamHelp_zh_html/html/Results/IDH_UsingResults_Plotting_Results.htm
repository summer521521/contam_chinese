绘制结果

ContamW 为您提供了一些用于绘制瞬态仿真结果的图表功能。您可以使用这些功能绘制气流、污染物和暴露结果。图表仅适用于当前加载的 CONTAM 项目的瞬态仿真结果。您可以绘制气流和压差、污染物浓度以及人员暴露和潜在剂量的结果。您可以从以下位置访问这些图表功能 S 模拟 菜单时，在 结果 模式。

注意： 如果在 ContamW 插入符号突出显示图标时选择绘图菜单项，则当显示关联的图表控制对话框时，该项目将出现在要绘制的项目列表中。此功能允许您通过在 SketchPad 上突出显示特定项目的结果来绘制它，而不是在图表控制对话框上的要绘制的项目列表中查找它。

绘制气流结果

菜单命令： S 模拟 → Plot A 气流结果...

绘制气流路径的气流和压差结果。您可以为那些实现双向气流元件的流路绘制双向流速。如果您选择不显示双向流动，ContamW 将绘制实现双向气流元件的气流路径的净流速。

绘制污染物结果

菜单命令： S 模拟 → Plot C 污染结果...

绘制房间污染物浓度的时间历史。

绘制曝光结果

菜单命令： S 模拟 → Plot E 曝光结果...

绘制居住者暴露于污染物的时间历史。

绘制粒子结果

菜单命令： S 模拟 → P 地段帕 r 标题结果...

绘制单个房间的粒径分布的时间历程。

绘制温度结果

菜单命令： S 模拟 → P 地块特 m 温度结果...

绘制多个房间/交汇处温度的时间历史。此功能用于绘制使用 TRNSYS 或 EnergyPlus 执行耦合传热计算时确定的温度结果。
