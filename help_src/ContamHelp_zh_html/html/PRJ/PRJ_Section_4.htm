第 4 部分：日间安排

日程表数据由 dschd_read() 函数读取并由 dschd_save() 函数保存。数据存储在 DY_SCHD 结构中，该结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。

当天的日程安排部分开始于：

_ndsch // 日程安排数 ( IX)

接下来是数据标题注释行，然后是所有数据 _ndsch 时间表。

对于每个时间表，第一数据行包括：

nr // 日程编号 ( IX ）；按从 1 到 _ndsch 的顺序

npts // 数据点数 (I2)

形状 // 0 = 矩形； 1 = 梯形 (I2)

utyp // 单位类型 (I2) {W}

ucnv // 单位转换 (I2) {W}

name[] // 日程名称 (I1) {W}

第二行有：

desc[] // 日程描述 (I1) {W} 可以为空

接下来是每个 npts 数据点的一行：

time // 一天中的时间 [s]（hh:mm:ss 转换为 I4）

ctrl //对应控制值(R4)[-]

该部分以以下内容结束：

-999 // 用于检查上述数据是否有读取错误
