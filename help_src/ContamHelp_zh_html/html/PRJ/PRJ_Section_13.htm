第 13 节：简单空气处理系统 (AHS)

AHS 数据由system_read( ) 函数读取并由system_save( ) 函数保存。数据存储在 AHS_DSC 结构中，该结构在 contam.h 中定义（对于 ContamW ），在 simdat.h 中定义 ContamX。

AHS 部分开头为：

_nahs // AHS 的数量 ( IX)

接下来是数据头注释行，然后是所有 _nahs 系统的数据。

对于每个 AHS，第一个数据行包括：

nr // AHS 数字 ( IX ）；按从 1 到 _nahs 的顺序

zone_r // 返回房间编号 ( IX)

zone_s // 供应区编号 ( IX)

path_r // 再循环空气路径编号 ( IX)

path_s //室外空气路径号( IX)

path_x // 排气路径号 ( IX)

color // 图标颜色 (I2) {W} {CONTAM 3.3}

第二行有：

desc[] // AHS 描述 (I1) {W} 可以为空

AHS 部分终止于：

-999 // 用于检查上述数据是否有读取错误
