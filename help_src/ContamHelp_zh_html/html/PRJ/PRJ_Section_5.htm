第 5 部分：周时间表

周计划数据由 wschd_read( ) 函数读取并由 wschd_save( ) 函数保存。数据存储在 WK_SCHD 结构中，该结构在 contam.h（针对 ContamW）和 simdat.h（针对 ContamX）中定义。

周计划部分开始于：

_nwsch // 周计划数 ( IX)

接下来是数据标题注释行，然后是所有 _nwsch 计划的数据。

对于每周计划，第一行数据包括：

nr // 日程编号 ( IX ）；按从 1 到 _nwsch 的顺序

utyp // 单位类型 (I2) {W}

ucnv // 单位转换 (I2) {W}

name[] // 日程名称 (I1) {W}

第二行有：

desc[] // 日程描述 (I1) {W} 可以为空

第三行有：

j1 … j12 // 12 天日程索引 ( IX ) – 转换为指针

该部分以以下内容结束：

-999 // 用于检查上述数据是否有读取错误
