第 8b 节：过滤器

过滤器数据由filter_read()函数读取并由filter_save()函数保存。数据存储在 FLT_DSC 结构中，该结构在 contam.h（针对 ContamW）和 simdat.h（针对 ContamX）中定义。

过滤器部分开始于：

_nfilt // 过滤器数量 ( IX)

接下来是所有数据 _nfilt 过滤器。

对于每个过滤器，第一数据行包括：

nr // 过滤器编号 ( IX ）；按从 1 到 _nfilt 的顺序

fe   // 过滤元素个数 ( IX)

纳米粒子   // 子元素数量，如果不是超级过滤器则为1 ( IX)

接下来是 纳米粒子 初始总相对载荷线：

load // 初始相对加载 [-] (R4)

负载   // 物种负载数 (I2) [0]

接下来是 加载 物种加载数据线：   [尚未使用； 负载 = 0]

    规格[]    // 物种名称 (I1)

    负载      // 初始相对加载 [-] (R4)
