第 22 条：暴露

曝光数据由 pexp_read( ) 函数读取并由 pexp_save( ) 函数保存。数据存储在 PEXP_DSC 结构中，该结构在 contam.h（针对 ContamW）和 simdat.h（针对 ContamX）中定义。

曝光部分开始于：

_npexp // 曝光次数 ( IX)

接下来是所有 _npexp 暴露的三行数据。

对于每次曝光，第一数据行包括：

nr // 曝光次数 ( IX ）；按从 1 到 _npexp 的顺序

如果产生污染物，gen // = 1 (I2)

ncg // 污染物代数 (I2)

cgmlt // 污染物产生乘数 [-] (R4)

注意 - 以下 5 项已从版本 3.2 的项目文件中删除：

inhmax // 峰值吸入率 [m^3/s] (R4) {未使用}

inhsch // 每周吸入时间表指数 (I2)

bodywt // 体重 [kg] (R4) {未使用}

u_inh // 吸入速率单位 (I2)

u_bw // 体重单位 (I2)

color // 图标颜色 (I2)

第二行有曝光描述：

desc[] // 曝光/人物描述 (I1)

第三行有12个占用表的索引：

odsch[12] // 每日占用时间表向量 – 12 个索引

接下来是污染物产生数据的 ncg 行：

名称 // 物种名称 (I1)

ps // 调度索引 (I2);转换为指针

cgmax // 峰值生成速率 [kg/s] (R4)

u_cg // 发电率单位 (I2) {W}

vf_type // 0=无值文件，1=使用 cvf，2=使用 dvf (I2) {CONTAM 3.2}

If vf_类型 > 0 然后该行继续包括：

   vf_node_name[NAMELEN] // 值文件节点名称 (I1) {CONTAM 3.2}

曝光部分终止于：

-999 // 用于检查上述数据是否有读取错误
