第 15 节：初始房间浓度

初始房间污染物质量分数由zone_read()函数读取并由zone_save()函数保存。质量分数存储在 contam.h 中为 ContamW 定义的 ZONE_DSC 结构中，并且它们存储在 ContamX 中的数组中。

房间污染物部分开始于：

nn // 后面的质量分数数 ( IX)

nn 应等于 _nzone * _nctm。

接下来是 _nzone 行，按从 1 到 _nzone 的顺序排列。

每个数据行包含按污染物顺序排列的 _nctm 质量分数 (R4)。

nr    // 房间编号 ( IX)

CC0[i][1] // 房间 i 污染物 1 的初始质量分数 (R4)

...

CC0[i][_nctm] // 房间 i 的初始质量分数（R4），污染物 _nctm

初始房间浓度部分终止于：

-999 // 用于检查上述数据是否有读取错误
