第 21 条：占用表

占用表由 oschd_read( ) 函数读取并由 oschd_save( ) 函数保存。数据存储在 OD_SCHD 结构中，该结构在 contam.h（针对 ContamW）和 simdat.h（针对 ContamX）中定义。

占用表部分开始于：

_nosch // 计划数量 ( IX)

接下来是数据标题注释行，然后是所有 _nosch 计划的数据。

对于每个时间表，第一数据行包括：

nr // 日程编号 ( IX ）；按从 1 到 _nosch 的顺序

npts // 点数 (I2)

u_XYZ // 位置坐标单位 (I2)

name[] // 日程名称 (I1) {W}

第二行有：

desc[] // 日程描述 (I1) {W} 可以为空

接下来是 npts 行计划数据：

time // 一天中的时间 [s]（hh:mm:ss 转换为 I4）

pz // 房间索引 (I2);转换为指针

X // 乘员的 X 坐标 [m] (R4)

Y // 乘员的 Y 坐标 [m] (R4)

relHt // 相对于当前水平面的高度 [m] (R4)

占用表部分终止于：

-999 // 用于检查上述数据是否有读取错误
