第 23 节：注释

注释由note_read()函数读取并由note_save()函数保存。数据存储在 contam.h 中定义的 NOTE_DSC 结构中。注释仅在 ContamW 中使用；在 ContamX 中读取此部分时，不会存储任何内容。

注释部分开头为：

_nnote // 注释数量 ( IX)

接下来是所有 _nnote 注释的数据：

对于每个注释数据行包括：

nr // 注释编号 ( IX ) {W};按从 1 到 _nnode 的顺序

color // 图标颜色 (I2) {W} {CONTAM 3.3}

note[] // 注释 (I1) {W}

注释部分以以下内容结束：

-999 // 用于检查上述数据是否有读取错误
