第 7 节：动力学反应

动力学反应数据由kinetic_read()函数读取并由kinetic_save()函数保存。数据存储在 KNR_DSC 结构中，该结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。

动力学反应部分开始于：

_nkinr // 动力学反应数 ( IX)

接下来是数据标题注释行，然后是所有 _nkinr 反应的数据。

对于每个反应，第一数据行包括：

nr // 反应数 ( IX ）；按从 1 到 _nkinr 的顺序

nkrd // 反应数 (I2)

name[] // 反应名称 (I1) {W}

第二行有：

desc[] // 反应描述 (I1) {W} 可以为空

接下来是 nkrd 行反应数据：

   prod[] // 产品品种名称（I1）

   src[] // 源物种名称 (I1)

   coef //反应系数(R4)

动力学反应部分终止于：

-999 // 用于检查上述数据是否有读取错误
