第 9 节：源/汇元件

源/宿元素由 cselmt_read( ) 函数读取并由 cselmt_save( ) 函数保存。数据存储在 CSE_DAT 结构中，这些结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。 CSE_DAT 包括指向元素特定数据结构的指针。

源/汇元素部分以以下内容开头：

_ncse // 源/汇元素的数量 ( IX)

接下来是数据头注释行，然后是所有 _ncse 元素的数据。

对于每个元素，第一个数据行包括：

nr // 元素编号 ( IX ）；按从 1 到 _ncse 的顺序

spcs[] // 物种名称 (I1)

ctype // 元素数据类型（字符串？I2）

  元素类型名称按 ctype 顺序存储在 _cse_dnames 中。

name[] // 元素名称 (I1) {W}

第二行有：

desc[] // 元素描述 (I1) {W} 可以为空

接下来是取决于元素数据类型的一行或多行数据。

接下来的介绍行给出了类型编号、定义为该编号的程序宏、类型名称 from_cse_dnames 以及保存数据的数据结构。数据结构在 ContamW 中的 celmts.h 和 ContamX 中的 selmts.h 中定义。只要 _csse_names 和相关数组中的顺序反映了该顺序，类型号就可以更改。

 

元素类型 0 [CS_CCF]“ccf”存储在结构 CSE_CCF 中。

常数系数源模型数据：

G // 生成速率 [kg/s] (R4)

D // 沉积速率[kg/s] (R4)

u_G // 生成单位 (I2) {W}

u_D // 沉积单位 (I2) {W}

 

元素类型 1 [CS_PRS]“prs”存储在结构 CSE_PRS 中。

压力驱动源数据：

G // 生成速率 [kg/s] (R4)

x // 压力指数 [-] (R4)

u_G // 生成单位 (I2) {W}

 

元素类型 2 [CS_CUT]“剪切”存储在结构 CSE_CUT 中。

浓度截止模型数据：

G // 生成速率 [kg/s] (R4)

Co // 截止浓度 [kg/kg] (R4)

u_G // 生成单位 (I2) {W}

u_C // 浓度单位 (I2) {W}

 

元素类型 3 [CS_EDS]“eds”存储在结构 CSE_EDS 中。

指数衰减模型数据：

G0 //初始生成速率[kg/s] (R4)

k // 衰减常数 [1/s] (R4)

u_G // 生成单位 (I2) {W}

u_k // 时间单位 (I2) {W}

 

元素类型 4 [CS_BLS]“bls”存储在结构 CSE_BLS 中。

边界层扩散模型数据：

hm //平均膜传质系数[m/s] (R4)

rho // 空气膜密度 [kg/m^3] (R4)

Kp //分配系数(R4)

M // 吸附剂质量/单位面积[kg/m^2] (R4)

u_h // hm 单位 (I2) {W}

u_r // rho (I2) {W} 的单位

u_M // M (I2) {W} 的单位

 

元素类型 5 [CS_BRS]“brs”存储在结构 CSE_BRS 中。

突发源数据：

M // 在一个时间步内添加到房间的质量 [kg] (R4)

u_M // 质量单位 (I2) {W}

 

元素类型 6 [CS_DVS]“dvs”   存储在结构体CSE_DVS中。

沉积速度汇模型数据：

dV        // 沉积速度 [m/s] (R4)

dA        // 沉积面积 [m^2] (R4)

u_V       // 速度单位 (I2) {W}

u_A       // 面积单位 (I2) {W}

 

元素类型 7 [CS_DRS]“drs”   存储在结构体CSE_DRS中。

沉积速率汇模型数据：

kd        // 沉积速率[1/s] (R4)

u_k       // 沉积速率单位 (I2) {W}


元素类型 8 [CS_DVR]“dvr”   存储在结构体CSE_DVR中。   {CONTAM 3.0}

沉积速度再悬浮源/汇模型数据：

dV        // 沉积速度 [m/s] (R4)

R         // 重新暂停率 [1/s] (R4)

dA        // 沉积表面积 [m2] (R4)

rA        // 重悬浮表面积 [m2] (R4)

u_V       // 沉积速度单位 (I2) {W}
u_R       // 重悬率单位 (I2) {W}
u_dA       // 沉积面积单位a (I2) {W}
u_dR       // 重悬浮面积单位 (I2) {W}

 

元素类型 11 [CS_PLM]“plm”   存储在结构 CSE_PLM 中。 {CONTAM 2.4b}

NRCC幂律模型数据：

a   // 系数(R4) [kg/s]

b   // 指数 (R4)

tp   // PL 模型时间 [s] (R4)

u_a  // 初始排放因子的单位 (I2) {W}

u_tp   // PL 模型时间单位 (I2) {W}

 

元素类型 12 [CS_PKM]“pkm”   存储在结构 CSE_PKM 中。 {CONTAM 2.4b}

NRCC峰值模型数据：

a   // 峰值发射率(R4) [kg/s]

b   // 拟合参数(R4)

tp   // 达到峰值的时间 [s] (R4)

u_a  // 峰值发射率单位 (I2) {W}

u_tp   // 高峰时间单位 (I2) {W}

 

元素类型 13 [CS_SUP]“sup”。 {CONTAM 2.4b}

超级源元素数据：

n   // 子元素的数量 (IX)

包含所有子元素编号列表的行：

埃尔姆特[]   // 子元素编号 (IX) – 转换为指针

源/汇元件部分以以下内容终止：

-999 // 用于检查上述数据是否有读取错误
