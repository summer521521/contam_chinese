第 10 节：气流元件

气流元件由 afelmt_read( ) 函数读取并由 afelmt_save( ) 函数保存。数据存储在 AFE_DAT 结构中，该结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。 AFE_DAT 包括指向元素特定数据结构的指针。

气流元件部分开始于：

_nafe // 气流元件数量 (IX)

接下来是数据头注释行，然后是所有 _nafe 元素的数据。

对于每个元素，第一个数据行包括：

nr // 元素编号 (IX);按从 1 到 _nafe 的顺序

icon // 用于表示流路的图标 (R4) {W}

dtype // 元素数据类型（字符串？I2）

// 元素类型名称按 dtype 顺序存储在 _afe_dnames 中。  

name[] // 元素名称 (I1) {W}

第二行有：

desc[] // 元素描述 (I1) {W} 可以为空

接下来是取决于元素数据类型的一行或多行数据。

接下来的介绍行给出了类型编号、定义为该编号的程序宏、来自 _afe_dnames 的类型名称以及保存数据的数据结构。数据结构在 ContamW 中的 celmts.h 和 ContamX 中的 selmts.h 中定义。只要 _afe_dnames 和相关数组中的顺序反映了该顺序，类型号就可以更改。

u_A // 面积单位 (I2) {X}

u_D // 直径单位 (I2) {X}
