第 16 节：气流路径

路径数据由path_read()函数读取并由path_save()函数保存。数据存储在 PATH_DSC 结构中，该结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。

路径部分开头为：

_npath // 路径数 ( IX)

接下来是数据头注释行，然后是所有 _npath 路径的数据。

对于每条路径，数据线包括：

nr // 路径号 ( IX）；按从 1 到 _npath 的顺序

flags // 气流路径标志值 (I2)

pzn // 房间 N 索引 ( IX ）；转换为指针

pzm // 房间 M 索引 ( IX ）；转换为指针

pe // 流元素索引 ( IX ）；转换为指针

pf // 过滤器索引 ( IX ）；转换为指针

pw // 风系数指数 ( IX ）；转换为指针

pa // AHS 索引 ( IX ）；转换为指针

ps // 调度索引 ( IX ）；转换为指针

pc // 控制节点索引 ( IX ）；转换为指针

pld // 级别索引 ( IX ）；转换为指针

X // 包络路径的 X 坐标 [m] (R4) {Contam 2.1}

Y // 包络路径的 Y 坐标 [m] (R4) {Contam 2.1}

relHt // 相对于当前水平面的高度 [m] (R4)

mult // 元素乘数 (R4)

wPset //恒定风压[Pa] (pw==NULL) (R4)

wPmod // 风速（？）修饰符（pw！= NULL）（R4）

wazm // 墙壁方位角 (pw!=NULL) (R4)

Fahs // AHS 路径流量 [kg/s] (pw==NULL) (R4)

Xmax // 流量或压力限制 - 最大值 (R4) {W}

Xmin // 流量或压力限制 - 最小值 (R4) {W}

icon // 用于表示流路的图标 (U1) {W}

dir // 画板上的正流向 (U1) {W}

color // 图标颜色 (I2) {W} {CONTAM 3.3}

u_Ht // 高度单位 (I2) {W}

u_XY // X 和 Y 的单位 (I2) {W}

u_dP // 压力差单位 (I2) {W}

u_F // 流量单位 (I2) {W}

vf_type // 0=无值文件，1=使用 cvf，2=使用 dvf (I2) {CONTAM 3.2}

vf_node_name[NAMELEN] // 值文件节点名称 (I1) {CONTAM 3.2}

cfd // cfd 路径 (0=否，1=是) (I2) {CONTAM 3.0}

如果 cfd = 1，则该行继续包含：{CONTAM 3.0}

name[] // cfd 路径 ID (I1)

cfd_ptype // 边界条件类型（0=质量流量，1=压力）(I2)

cfd_btype // 压力 bc 类型（如果 ptype = 1，0=线性，1=停滞压力）(I2)

cfd_capp // 耦合方法（1=压力-压力）(I2)

路径部分以以下内容终止：

-999 // 用于检查上述数据是否有读取错误
