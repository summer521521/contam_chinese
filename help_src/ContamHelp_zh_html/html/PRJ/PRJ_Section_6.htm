第 6 节：风压剖面

风压剖面数据由wind_read()函数读取并由wind_save()函数保存。数据存储在 WIND_PF 结构中，该结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。

风压剖面部分开始于：

_nwpf // 风压剖面的数量 ( IX)

接下来是数据头注释行，然后是所有 _nwpf 配置文件的数据。

对于每个风压剖面，第一数据行包括：

nr // 个人资料编号 ( IX ）；按从 1 到 _nwpf 的顺序

npts // 数据点数 (I2)

类型 // 1 = 线性； 2 = 三次样条； 3 = 三角函数 (I2)

name[] // 日程名称 (I1) {W}

第二行有：

desc[] // 配置文件描述 (I1) {W} 可以为空

接下来是每个 npts 数据点的一行，用于计算风压剖面的三次样条或三角系数：

   azm[] // 风方位角值 {R4} [度]

   coef[] // 归一化风压系数 {R4} [-]

该部分以以下内容结束：

-999 // 用于检查上述数据是否有读取错误
