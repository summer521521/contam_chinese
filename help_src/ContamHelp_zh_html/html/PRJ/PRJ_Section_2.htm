第 2 部分：物种和污染物

物种/污染物数据由 spcs_read( ) 函数读取并由 spcs_save( ) 函数保存在 ContamW 中。它们由 ContamX 中的 ctm_read( ) 读取。

物种/污染物部分开头为：

_nctm // 模拟的物种总数（=污染物）（I2）

下一行列出了被视为污染物的 _nctm 物种的数量。污染物的顺序很重要，在 ContamW 中设置。

_ctm[i] // i = 0 到 nctm-1

本节第三行给出：

_nspcs // 物种数量 ( IX)

接下来是标题行，然后是每个物种的两行数据：

第一行物种数据包括：

nr // 物种编号 ( IX )，按从 1 到 _nspcs 的顺序

sflag // 1 = 模拟，0 = 非模拟物种 (I2) {W}

ntflag // 1 = 非痕量，0 = 痕量物种 (I2) {W}

molwt // 摩尔质量 [kg/kmol] - 气体 (R4)

mdiam // 平均直径 – 颗粒 [m] (R4)

edens // 有效密度 – 粒子 [kg/m^3] (R4)

衰减 // 衰减常数 [1/s] (R4) {W}

Dm //分子扩散系数[m 2 /s] (R4)

ccdef // 默认浓度 [kg/kg 空气] (R4)

Cp //（未使用）恒压比热[J/kgK] (R4)

Kuv // UVGI 磁化率常数 [m2/J] (R4) {CONTAM 3.2}

ucc // 显示浓度的单位 (I2) {W}

umd // 显示平均直径的单位 (I2) {W}

ued // 显示有效密度的单位 (I2) {W}

udm // 显示扩散系数的单位 (I2) {W}

ucp // 显示比热的单位 (I2) {W}

name[] // 物种名称 (I1)

第二行是：

desc[] // 物种描述 (I1) {W}

该部分以以下内容结束：

-999 // 用于检查上述数据（I2）中是否存在读取错误
