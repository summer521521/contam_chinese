第 19 节：管道段

管道数据由duct_read()函数读取并由duct_save()函数保存。数据存储在 DCT_DSC 结构中，这些结构在 contam.h（对于 ContamW）和 simdat.h（对于 ContamX）中定义。

风管段部分开始于：

_ndct // 管道数量 ( IX)

接下来是数据标题注释行，然后是所有 _ndct 管道的数据。

对于每个管道，数据行包括：

nr // 管道编号 ( IX ）；按从 1 到 _ndct 的顺序

flags // 风道标志值 (I2)

pjn // 结点 n 索引 ( IX ）；转换为指针

pjm // 路口 m 索引 ( IX ）；转换为指针

pe // 风道流元素索引 ( IX ）；转换为指针

pf // 过滤器索引 ( IX ）；转换为指针

ps // 调度索引 ( IX ）；转换为指针

pc // 控制节点索引 ( IX ）；转换为指针

dir // 画板上的正流向 (U1) {W}

length // 风管段的长度 [m] (R4)

Ain // 入口端的流动面积 [m^2] - 未来 (R4)

Aout // 出口端的流通面积 [m^2] - 未来 (R4)

sllc // 局部损失系数之和 (R4)

color // 图标颜色 (I2) {W} {Contam 2.4}

u_L // 长度单位 (I2) {W}

u_A // 流通面积单位 (I2) {W}

vf_type // 0=无值文件，1=使用 cvf，2=使用 dvf (I2) {CONTAM 3.2}

vf_node_name[NAMELEN] // 值文件节点名称 (I1) {CONTAM 3.2}

管道段部分终止于：

-999
