重悬模型沉积

该源/汇模型通过沉积到表面来去除房间空气中的污染物，而源排放则通过表面的再悬浮来实现。总沉积表面积保持恒定，由 ( 多 ·A s ) 项在下面的等式中，但重悬浮率和面积可以通过时间表或控制值进行修改。 单位以默认计算单位提供。

Ra(t) = nd ( 多 ·A s) rair(t) Ca(t)

Sa(t) = （ctrl· r Ar) La(t)

其中：

Ra(t ) = 污染物去除率（通过沉积） a 一次从房间空气到地面 t [kga/s]

vd  = 沉积速度 [m/s]

As = 沉积表面积 [m 2]

ρair(t) = 某时刻源区空气密度 t [kgair/m3]

Ca(t ) = 污染物浓度 a 某时间房间风量 t [kga/kgair]

Sa(t) = 污染物来源（再悬浮）率 a 一次从地表到房间空气 t [kga/s]

r = 重悬率 [1/秒]

Ar = 再悬浮表面积 [m 2]

La(t ) = 污染物浓度 a 时间 t 时沉积表面上的重量 [kg a/m2]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

沉积速度： 输入沉积速度和单位。空气密度将在模拟时从源/汇所在房间获得。

沉积表面积： 输入所需的表面积和单位。您可以输入值 1.0 并使用源/汇乘数来指定该元素的每个特定实例的总沉积表面积。

重新暂停率： 输入重新悬浮率和单位。

重悬表面积： 再悬浮表面房间可以不同于沉积表面房间，例如，鞋撞击沉积表面。一旦与源/汇相关联，就可以使用也与源/汇相关联的时间表或控制来改变重悬浮率。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
