源/汇 - 乘数和位置属性

源/汇数量： 这是当您突出显示图标时出现在状态栏中的数字。

乘数： 模拟期间将乘以源强度的常数值。使用此功能，您可以定义一个 源/汇元件 具有每单位面积的源强度，然后使用乘数作为使用每单位面积源/汇元件的每个源/汇的房间的面积。

注：乘数用作边界层扩散模型的表面积，应以 m 为单位输入 2.

初始浓度： 一些源/汇是可逆的，因为它们能够在表面内或表面上储存并从表面重新排放或重新悬浮污染物。这些源/汇实际上为污染物传输方程贡献了污染物节点。目前，有两种这样的元素类型： 边界层扩散控制 and 重悬沉积 。 的 初始浓度 将为这些元素类型启用属性，您可以设置这些初始载荷的值和单位。

颜色: 设置图标的颜色。这允许您区分 SketchPad 上的不同图标。通过以下方式激活图标颜色的显示 V 瞧—— > 图标 C 着色 菜单项。

地点： 这些是所选源/汇的 X、Y 和相对高程 (Z) 坐标和单位。 X 和 Y 是绝对坐标，相对高程是相对于源/汇所在水平面的。要创建点源，请将最小值和最大值设置为相同。要创建沿着房间的一维轴分布排放的源，请输入跨越源所在房间的所需房间的最小值和最大值。

这些值对于位于 a 内的任何源/汇都是必需的。 一维对流/扩散区 （见 一维房间数据 ）。这些坐标将由 ContamW 对照其所在的 1D 房间的轴进行验证。当您选择以下任一选项时，将自动执行此验证 R 联合国模拟 or 创建一个Contam X 输入文件 命令来自 S 模拟 菜单。

CFD  来源ID： 位于 CFD 房间内的每个源/汇都必须被赋予唯一的标识符。此 ID 将用于标识 CFD0 编辑器中的源/汇，该编辑器用于定义相对于 CFD 房间的接口路径的位置。
