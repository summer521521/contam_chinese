使用源和接收器

源和汇用于在房间内产生污染物并从房间中去除污染物。源可用于模拟建筑材料（例如地毯、家具）和偶发事件（例如清洁产品的使用）。您可以在项目中定义几乎无限数量的源。

要实现源/汇，您必须首先定义要与源/汇关联的物种。然后，将源/汇图标放置在 SketchPad 上的房间内。将源/汇放置在 SketchPad 上后，您可以定义它们的特征，移动、复制和删除它们。您必须将每个源/汇与预定义的物种相关联，并提供描述其排放和/或去除特征的具体信息。

注意： 值得注意的是，即使您将一个物种与每个源/汇相关联，在执行模拟时，实际上只会使用那些被指定为污染物的物种（请参阅 物种特性 ).
