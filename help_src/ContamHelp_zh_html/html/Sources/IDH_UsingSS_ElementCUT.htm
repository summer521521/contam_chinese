源/汇元件：截止浓度模型

对于挥发性有机化合物，源模型有时使用截止浓度模型来表示。

Sa(t) = 多 ·ctrl· G (1 - Ca(t)/Ccut)

Sa (t) = 时源强度 t [kga /s]

G = 生成率 [kg a/s]

Ca(t) = 污染物浓度 a 时间在该区域 t [kga/kgair]

Ccut = 排放停止时的截止浓度

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

发电率： 该系数描述了污染物进入该区域的速率，作为浓度的函数。

截止浓度： 源停止排放时的浓度水平。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
