源/汇元件：边界层扩散模型

具有线性吸附等温线的边界层扩散控制可逆汇/源模型遵循[ 阿克斯利 1991 ]。边界层是指材料表面上方的房间，近表面浓度与气相浓度之间存在浓度梯度。污染物转移到表面（水槽）上的速率由以下方程定义。单位以默认计算单位提供。

Sa(t) = 控制键 ·h·d·A s[Ca (t)- Cs(t)/k]

其中：

h   = 水槽上的薄膜传质系数 [m/s]
d   = 空气膜密度 [kg air/m3]
As   = 吸附剂的表面积 [m 2]
Ca  = 空气中浓度 [kg a/kgair]
Cs = 吸附剂中的浓度
k   = 亨利吸附常数或分配系数 [公斤 air/kg 表面 ]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

薄膜传输系数： 吸附剂表面上方边界层（或薄膜）内污染物的平均传质系数。

空气膜密度： 这是空气体积密度和近表面密度的平均值。

表面质量： 输入单位表面积的质量，并使用源/汇乘数输入吸附剂的总面积。

分配系数： 将大量空气中的污染物浓度与吸附材料表面的污染物浓度联系起来的系数。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。

注：表面积，As ，由源/汇乘数提供，单位为 m 2.
