源/汇元件：压力驱动模型

压力源/汇模型旨在模拟由内部和外部房间之间的压差控制的污染物源，例如氡气或土壤气体进入。   单位以默认计算单位提供。

Sa(t) = 多 ·ctrl· G (DP)n

S    = 某时刻污染物源强度 t
G   = 发电率系数 [kg a/Pan ·s]
DP = 压差（环境压力 - 房间压力）[Pa]
n    = 压力指数 [无单位]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

发电率： 该系数描述了污染物进入该区域的速率，作为压差的函数。

压力指数： 用于描述污染物进入压力的依赖性。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
