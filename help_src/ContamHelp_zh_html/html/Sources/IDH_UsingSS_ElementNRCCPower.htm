NRCC – 幂律模型

这是一个基于经验的模型，旨在与 NRCC 材料排放数据库兼容 [ 霍华德-里德 2004 and 2006]. 

For t £ tp 初始排放因子为

So(t) = 多 ·一个 tp-b

并为 t > tp

S(t ) = 乘数 ·a t -b

注意： 每次模拟只能触发一次此类源。如果使用计划，第一个非零计划值将触发源，否则它将立即开始发射。

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

初始排放因子： 输入初始排放因子 S0 及其单位 [kg/s]。

系数： 这对应于 a 在上面的等式中。它将根据 S0, tp and b 。的价值 a 将由 NRCC 数据库提供并由 CONTAM 存储，但提供初始排放因子作为定义模型的更直观方法。

指数： 输入指数值 b 在上面的等式中。

PL 模型时间： 输入时间 tp 在此之前，初始排放率适用。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
