源/汇元件：常数系数模型

这是一个通用的源/汇元件，可直接输入污染物的产生和去除率。单位以默认计算单位提供。

Sa(t) = 多 ·ctrl· G - 多 ·ctrl· R Ca(t)

G = 生成率 [kg a/s]

R = 有效去除率 [kg air/s]

通过将污染物的一阶去除率 [1/s] 乘以空气密度 [kg] 来确定air/m3 ] 和体积 [m 3 ] 源所在的房间。

Ca(t) = 污染物浓度 a 在房间 [kg a/kgair]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

发电率： 污染物进入该区域的速率。

去除率： 从房间中去除污染物的速率。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
