NRCC - 峰值模型

这是一个基于经验的模型，旨在与 NRCC 材料排放数据库兼容 [ 霍华德-里德 2004 and 2006].

Sa(t ) = 乘数· a ·exp{-0.5[(1/ b)·ln(t/tp)]2}

其中：

a, b and tp 是通常根据发射室测量确定的经验系数。
排放率， Sa(t ) [公斤 a/s] ，将增加直到时间 tp 此时它将开始腐烂。

mult = 源/汇乘数 [无单位]

注意： 每次模拟只能触发一次此类源。如果使用计划，第一个非零计划值将触发源，否则它将立即开始发射。

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

峰值排放因子： 输入峰值排放因子 a 。这就是价值 S(t) 当时间 t = tp.

配件参数： 输入参数值 b 在上面的等式中。

高峰时间： 输入时间 tp 达到峰值排放因子。

说明： 用于输入特定源/汇元素的更详细描述的字段。
