沉积速度汇模型

该汇模型是作为使用沉积速度创建汇的便捷方法而提供的。 单位以默认计算单位提供。

Ra(t ) = 乘数 · 控制键 ·nd As rair(t) Ca(t)

其中：

Ra(t ) = 某时间的去除率（通过沉积） t [kga/s]

vd  = 沉积速度 [m/s]

As = 沉积表面积 [m 2]

ρair(t) = 某时刻源区空气密度 t [kgair/m3]

Ca (t ) = 污染物浓度 a 在某个时间 t [kga/kgair]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

该模型与沉积速率汇模型的不同之处在于，它不依赖于其所在房间的空气质量。

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

沉积速度： 输入沉积速度和单位。空气密度将在模拟时从源/汇所在房间获得。

表面积： 输入所需的表面积和单位。您可以输入值 1.0 并使用源/汇乘数来指定总沉积表面积。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
