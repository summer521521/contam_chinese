源/汇元件：衰减源模型

在模拟过程中，可以使用时间表或控制信号多次启动和停止衰减源模型。将计划或控制值设置为零将关闭源，而计划或控制值大于零将重新打开源。如果没有计划与此源接收器关联，它将立即激活。

Sa(t) = 多 ·ctrl键 ·G 0 e-Dt/tc

其中：

Sa (t) = 污染物来源强度 a 在某个时间 t [kga/s]

G0 = 初始生成率 [kg a/s]
Dt    = 自发射开始以来经过的时间 [s]
tc   = 时间常数 [s]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

初始发电率： 污染物的产生由时间表控制。当计划从零变为非零值（在零和一之间）时，污染物开始生成。初始生成率等于调度值乘以初始生成率。单个时间表可用于在不同时间启动多个排放。

衰减常数： 1/t c，生成速率衰减原始速率0.368的时间。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
