定义源和汇

当你画了一个 源/汇图标 在 SketchPad 上，您必须使用图标定义过程来定义它（请参阅 定义建筑组件图标 ) 显示和编辑属性。这将显示“Source/Sink Properties”属性表。所有源/汇属性的详细描述在 源和汇属性 本手册的部分。定义属性后，图标将显示为黑色。从现在开始，您可以使用此图标访问源/汇的属性。

与源/汇相关的属性是可选的 乘数 and 时间表 信息。您可以使用乘数按比例增加特定源/汇污染物的产生或去除。例如，如果您创建了一个 源/汇元件 对于每平方米表面积（例如地板覆盖物）的特定排放率，您可以使用此乘数来计算在其中放置源/汇实现此功能的每个房间的源/汇的表面积 源/汇元件 。你可以使用 时间表 随着时间的推移改变源/汇的发射/去除特性。例如，您可以提供 时间表 对于在执行瞬态模拟时会定期导致污染物释放的偶发源（突发源）。
