源/汇 - 源/汇元素属性

源/汇数量： 这是当您突出显示图标时出现在状态栏中的数字。 ContamW 自动将此编号分配给每个 源/汇 一旦它们被定义。当您向项目添加源/汇或从项目中删除源/汇时，此数字可能会发生变化。每次保存项目时，都会重新分配源/汇编号。编号从顶层开始分配，从 SketchPad 的左上角开始，从 SketchPad 的从左到右、从上到下移动。

源/汇元件： 源/汇元件提供与所选源/汇相关联的数学模型。

说明： 显示在“名称”字段中选择的元素的描述属性。

姓名： < 必填 > 从名称列表中包含的源/汇元素中选择一个源/汇元素。您可以通过在项目中创建新元素或从库文件导入它们来将元素添加到列表中。使用“新元素”按钮定义新元素。激活 CONTAMW 库管理器，通过“库”按钮从库文件导入元素（请参阅 使用数据和库 ）。使用“编辑元素”按钮可以修改“名称”字段中显示的元素的属性。
