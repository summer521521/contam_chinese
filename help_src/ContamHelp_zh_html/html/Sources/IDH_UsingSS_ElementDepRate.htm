沉积速率汇模型

该水槽模型是作为使用沉积速率创建水槽的便捷方法而提供的。该模型取决于源/汇所在房间中的空气质量，并且其行为类似于具有相同反应速率的动力学反应。 单位以默认计算单位提供。

Ra(t ) = 乘数 ·ctrl键 ·kd Vz rair(t) Ca(t)

其中：

Ra(t ) = 某时间的去除率（通过沉积） t [kga/s]

Vz = 房间体积 [m 3]

kd  = 沉积速率 [1/s]

ρair(t) = 某时刻源区空气密度 t [kgair/m3]

Ca (t ) = 污染物浓度 a 在某个时间 t [kga/kgair]

mult = 源/汇乘数 [无单位]

ctrl = 计划或控制值 [无单位]

姓名： 输入要用于标识源/汇元素的唯一名称。该元素将保存在当前项目中，并且可以与多个源/汇关联。

沉积率： 输入沉积速率和单位。体积和密度将在模拟时从源/汇图标所在的房间获得。

品种： 从下拉框中选择先前定义的物种。

说明： 用于输入特定接收器/源元件的更详细描述的字段。
