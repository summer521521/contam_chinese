创建物种和污染物

您可以通过激活来创建物种 项目种类 对话框通过 Data → C 污染物… 菜单或使用 CONTAMW 数据和库管理器：通过以下方式访问物种对话框： Data → Spe 情况… 菜单。 “项目物种”对话框将使您能够为当前项目创建和修改物种。图书馆管理器将使您能够从物种相关的图书馆导入物种。每个物种/污染物特性的详细描述在 物种特性 本手册的部分。每个物种都有一个 默认浓度 与您定义的关联（或保留零）。您可以使用此默认浓度来设置建筑物内的初始浓度，而无需使用源元素。该默认值也将用作环境污染物浓度，除非被污染物文件或 WPC 文件中的值覆盖（请参阅 使用控件 and 使用控件 ).
