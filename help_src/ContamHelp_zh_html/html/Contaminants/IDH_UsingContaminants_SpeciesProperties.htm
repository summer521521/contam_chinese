物种特性

本节提供特定物种属性的详细描述。

姓名： 您将在整个程序中使用这个名称来指代该物种。您必须为每个物种指定当前项目唯一的名称。

Molar 弥撒 : 摩尔质量，单位为 g/mol 或 kg/kmol。 CONTAM 使用该值将体积浓度转换为质量分数，并计算由非痕量污染物组成的空气混合物的气体常数。您必须输入至少 1.0 的值。

默认浓度： CONTAM 将应用此值作为您创建的每个房间的默认初始浓度。您可以根据您的要求修改各个房间的初始浓度。这也是模拟期间的环境污染物浓度，不使用外部污染物数据文件（CTM 或 WPC）之一的污染物数据。

扩散系数： 该值将在模拟过程中使用 一维管道模型 的 短时间步长法 （参见 运行控制属性 在 使用模拟 部分）。扩散系数将根据给定管道段中的雷诺数从此值计算。

平均直径： 如果该物种是颗粒类型物种，请输入平均粒径。 CONTAM 在颗粒计数单位与颗粒质量和体积单位之间进行转换时使用此值，并确定简单颗粒过滤器元件对于此尺寸颗粒的过滤效率（请参阅 简单的粒子过滤器 在 使用过滤器 部分）。

注：这不一定是空气动力学直径，因为当前污染物源模型不处理与气态污染物不同的颗粒。

有效密度： 输入您希望 CONTAM 用作您想要视为颗粒类型物种的物种的有效密度的密度。目前，CONTAM 仅在粒子计数单位与粒子质量和体积单位之间转换时使用此值。

比热： 仅当以下情况时，才会在模拟过程中使用污染物的比热： 可变结温 方法被选择。

衰减率： 这是基于放射性物质半衰期的指数衰减常数，使用以下公式计算：

衰减率 = ln(2) / t1/2,

哪里 t1/2 是放射性物质的半衰期（以秒为单位）。

ContamW 将使用该值来执行放射性污染物的单位转换。放射性物质与其他物质的区别仅在于与其相关的单位，而不是通过对放射性衰变过程进行建模。要模拟放射性衰变，请使用动力学反应（参见 动力学反应数据 ).

UVGI 敏感性常数： 用于确定 UVGI 的物种相关速率常数   filter effectiveness for a specific contaminant [m 2 /J]。 [中提供了几个物种的值 科瓦尔斯基等人。 2000年 ] 范围从 10 -3 cm2/m J 至 10 -6 cm2/m J.（3.2 版新增）

无痕量污染物： 非痕量污染物是那些可以影响空气密度的污染物，即被视为空气的组成部分。 （参见 无痕污染物和水蒸气 )

在模拟中的使用： 这决定了一个物种是否会成为污染物。选中“在模拟中使用”表示该物种将成为污染物并在模拟中使用。

说明： 用它来提供该物种的更详细描述。

指导值： 目前尚未实施。

指南说明： 目前尚未实施。
