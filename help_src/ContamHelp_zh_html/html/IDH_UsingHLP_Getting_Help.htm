寻求帮助

本帮助手册有印刷版和 ContamW 程序随附的“在线”帮助。有多种方法可以访问在线帮助系统。您可以使用 H 帮助内容... 选择的 Help 菜单显示帮助手册的内容或使用上下文相关功能访问帮助。

帮助内容

您可以浏览帮助系统的内容或索引来查看您选择的主题，并使用搜索功能显示包含您感兴趣的关键字的帮助主题列表。

上下文相关帮助

上下文相关帮助功能旨在为您提供与当前活动的 ContamW 窗口或对话框相关的特定信息。要激活上下文相关帮助系统，请按 F1 关键。

帮助索引和搜索

如果您在查找特定主题时遇到困难，请务必使用在线帮助索引和搜索功能。

获得技术支持

如果您需要联系 CONTAM 开发人员以获取有关使用该程序的指导或报告该程序的技术问题，您可以通过下面提供的链接中的在线论坛进行联系。

https://groups.google.com/a/list.nist.gov/forum/#!forum/contam
