占用者日时间表属性

这些是您用来定义占用者日程安排的属性和方法。

姓名： 这是您为该占用者的日程安排指定的名称。该名称在此项目内的占用者日程表中必须是唯一的。

说明： 用于输入具体时间表的更详细描述的字段。

时间表数据： 这些是您设置用于定义占用日计划的时间/位置数据点。时间以 hh:mm:ss 格式输入，并且必须提供前导零和尾随零；例如，上午 9:30 将输入为 09:30:00，而不是 9:30:00。该位置由居住者在指定时间段内居住的楼层和房间组成。从房间列表中选择“null”值，表示在给定时间段内居住者不在建筑物内。

如果居住者进入一维对流扩散房间，位置数据还将包括房间坐标。对于这种情况，请输入占用者进入的每个 1D 房间的 X、Y 和相对高程 (Z) 坐标和单位。 X 和 Y 是绝对坐标，相对标高是相对于人员进入的每个一维房间的水平面。

插入新数据点
在“修改数据”块中输入新的时间和位置数据，按“ << 插入 << " 按钮，数据点将以正确的顺序插入到明细表中。

修改现有数据点
从列表中选择要修改的数据点，输入修改后的数据，然后按“ << 插入 << " 按钮覆盖先前输入的数据点。

删除现有数据点
从列表中选择要删除的数据点，然后单击“删除”按钮。您无法删除 00:00:00 或 24:00:00。
