日程安排属性

这些是您用来定义日程安排的属性和方法。您可以通过按“以图形方式显示”按钮以图形方式显示时间表。

姓名： 这是您为这一天的日程安排指定的名称。该名称在项目内的日程安排中必须是唯一的。

说明： 用于输入具体时间表的更详细描述的字段。

形状： 使用此复选框可设置 CONTAMW 如何解释您定义的数据点之间的值。

  • 矩形：以数据点之间的阶跃函数为特征
  • 梯形：特点是数据点之间进行线性插值。

时间表数据： 这些是您设置用来定义日程安排的时间/乘数对。时间以 hh:mm:ss 格式输入。

对于无单位调度，该值是 0.0 到 1.0 之间的乘数，用于调低各种可调度建筑组件的最大设计值，例如风扇流量元件的恒定体积流量。

插入新数据点
在对话框的编辑计划值部分的“时间”和“值”编辑框中输入新的时间/乘数对，然后按“ << 插入 << “按钮。数据点将以正确的顺序插入到明细表中。

修改现有数据点
从列表中选择要修改的数据点，输入修改后的数据，然后按“ << 插入 << " 按钮覆盖先前输入的数据点。

删除现有数据点
从列表中选择要删除的数据点，然后单击“删除”按钮。您无法删除 00:00:00 或 24:00:00。
